# currency

CLI currency converter

## Usage

### Commands

#### positional arguments

input&emsp;&emsp;input in format of "100 eur to usd"

#### options

-h, --help&emsp;&emsp;show help message
-l, --list&emsp;&emsp;return list of available currencies

### Windows

```powershell
$ py currency.py "49.99 gbp to rub"
> 49.99 GBP to RUB = 5362.63 RUB
```

### Other

download source code (or git clone) and run currency.py

```bash
$ py currency.py " " --list
> All available currencies:
> AED: UAE Dirham
> AFN: Afghan Afghani
> ...
```
